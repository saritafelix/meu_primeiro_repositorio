# 🚀 Guia rápido GitHub para iniciantes

Este repositório reúne comandos básicos que estou aprendendo no Git e GitHub.

## 📦 Criando um repositório local

```bash
git init
📄 Adicionando arquivos
git add .
💾 Salvando alterações
git commit -m "primeiro commit"
☁️ Conectando ao GitHub
git remote add origin URL_DO_REPOSITORIO
📤 Enviando arquivos
git push -u origin main
📥 Atualizando projeto
git pull
🔎 Verificando alterações
git status
